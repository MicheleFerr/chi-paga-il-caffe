# Chi Paga il Caffè? ☕

Un gioco PWA dove chi indovina il numero segreto... paga il caffè!

## Come si gioca

1. Scegli la difficoltà (range del numero segreto)
2. A turno, i giocatori inseriscono un numero
3. Il range si restringe ad ogni turno
4. Chi indovina il numero segreto **paga il caffè**!

## Sviluppo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Il progetto è configurato per Vercel. Basta collegare il repository GitHub.

## Tech Stack

- SvelteKit 2
- Svelte 5
- PWA con @vite-pwa/sveltekit
- Vercel Adapter
